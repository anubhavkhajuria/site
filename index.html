About Me

My research interests lie at the intersection of High-Performance Computing, neural network optimization, and compiler technologies for deep learning. While frameworks like PyTorch simplify model development, they often abstract away critical performance details.

I am passionate about bridging this gap by leveraging compiler infrastructures like MLIR to automatically generate efficient code for heterogeneous hardware, especially GPUs. By designing modular compiler passes, such as operator fusion, progressive lowering, and loop/kernel transformations, I aim to improve the runtime performance of deep learning models without requiring changes to user code.

My ultimate goal is to build a transparent and portable compilation pipeline that delivers near-peak hardware performance for neural networks, enabling scalable and energy-efficient execution across diverse platforms.
